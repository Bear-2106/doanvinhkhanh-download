# QR-DL

Thu muc nay dung de tap trung cac file can upload len GitHub Pages.

Sau khi build app Android, chay:

```powershell
.\Sync-QRDL.ps1
```

Script se:

- build APK Release tu mobile project `DoAnVinhKhanh`
- tim file APK moi nhat
- copy thanh `QR-DL\DoAnVinhKhanh.apk`
- cap nhat version query trong `QR-DL\index.html`

Sau do ban chi can upload 3 file:

- `DoAnVinhKhanh.apk`
- `index.html`
- `.nojekyll`
